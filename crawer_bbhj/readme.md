# 宝贝回家未成年人失踪数据爬取与分析工具

本项目提供图形化界面，用于爬取“宝贝回家”公益网站（so.baobeihuijia.com）公开的失踪人员信息，自动筛选出**失踪时未满 18 周岁**的案例，并存储到本地 SQLite 数据库中。同时支持重复记录检查，便于后续数据清洗与分析。

## 功能特点

- 🕷️ 支持按类别爬取：全部 / 家寻宝贝 / 宝贝寻家 / 其他寻人 / 海外寻亲 / 烈士寻根
- 📄 可设定爬取页数范围及请求间隔（**强烈建议间隔 ≥ 1 秒**，以尊重网站服务器）
- 🔍 自动计算失踪时周岁年龄，仅保留 `年龄 < 18` 的记录
- 🗄️ 数据保存至 `bbhj_under18.sql` 数据库，表名 `宝贝回家未成年`
- 📊 内置数据查看窗口，支持表格浏览
- ✅ 重复记录检测脚本（按姓名+生日+失踪时间、姓名+生日）

## 文件结构及说明

| 文件名 | 作用 |
|--------|------|
| `beaubei_crawler_gui.py` | 主程序入口，基于 PySide6 构建 GUI 爬虫，负责界面交互、数据抓取与入库 |
| `crawler_main_ui.py` | 主窗口 UI 定义（由 PyQt5 UI 转换），包含爬虫参数输入、按钮、进度条和日志框 |
| `data_viewer_ui.py` | 数据查看窗口 UI 定义，提供 `QTableWidget` 表格展示数据库内容 |
| `duplicate_checker.py` | 重复记录检测脚本，连接数据库并输出重复项，辅助数据质量校验 |
| `requirements.txt` | 项目依赖清单，包含 PySide6、Requests、jieba、wordcloud 等库 |
